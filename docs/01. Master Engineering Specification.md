# ??? RentGuard AI — Master Engineering Specification
## Version 1.0 — MVP Governance Edition

### 1. Objetivo
Este documento define las reglas obligatorias para la implementación del MVP. El objetivo es construir un sistema mantenible, modular y operacionalmente simple.

### 2. Arquitectura Base
- **Frontend:** Angular SPA (Standalone + Signals).
- **Backend:** Modular Monolith (Vertical Slice).
- **Core:** RentGuard.Core.Business (Private Core Submodule).
- **Data:** SQL Server + EF Core.
- **Integration:** SQL Outbox Worker.

### 3. Restricciones Críticas
- Prohibido Microservicios (complejidad prematura).
- Prohibido Event Sourcing (sobreingeniera para MVP).
- Obligatorio Tenant Filtering en todas las capas.
